Инструкция
1. Установить зависимости: pip install -r requirements.txt
2. Запустить: uvicorn main:app --reload
3. Открыть http://127.0.0.1:8000/login
4. По умолчанию создан диспетчер: username=dispatcher password=dispatcher
5. Диспетчер может создавать пользователей (буровиков). Буровик при входе увидит форму.
